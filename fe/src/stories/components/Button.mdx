# Button Component

O componente `Button` é utilizado para renderizar botões em diferentes variantes e tamanhos, oferecendo flexibilidade para diversos cenários de uso.

## Propriedades

### Variantes

- **default**: Estilo padrão do botão.
- **destructive**: Estilo para ações destrutivas.
- **outline**: Botão com borda, sem preenchimento.
- **secondary**: Estilo secundário.
- **ghost**: Botão fantasma, sem preenchimento ou borda.
- **link**: Estilo de link, sem preenchimento ou borda.

### Tamanhos

- **default**: Tamanho padrão.
- **sm**: Botão pequeno.
- **lg**: Botão grande.
- **icon**: Botão para ícones.

### Propriedades Adicionais

- **asChild**: Renderiza o botão como um elemento filho de outro componente.
- **disabled**: Desabilita o botão.
